# Tejas Deshkulkarni Portfolio

Static, Netlify-ready portfolio site for Tejas Deshkulkarni.

## Local Preview

Open `index.html` directly in a browser, or run a tiny static server:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Deploying To Netlify

Use the repository root as the site directory. The included `netlify.toml` publishes the current folder with no build command.

## Included Sections

- Home
- About
- Education
- Skills
- Projects
- Resume
- Contact
- Data and website development service pages
